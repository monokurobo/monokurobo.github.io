买房记

发表于 2018-10-24 分类于 生活

背景

  来了3年多了，才渐渐的适应了这边的生活节奏，饮食习惯，行为操守。这个时候中国人的传统思想就又出来作祟了-“老租房子也不是个事啊，到老了，房子也不是自己的”。再加上一些中国朋友的怂恿，老婆的鼓动，一场浩浩荡荡的买房之旅就开始了。

总体介绍

  说起日本的房子，一般就是两种，公寓或者一户建，各有利弊，对比如下：

类型 优点 缺点
公寓 管理简单，离车站比较近，生活便利设施多，转手好卖 每个月要交管理费，房屋修缮费，停车费
一户建 随便跑跳蹦，面积大，每个月不用交乱七八糟的费用 自己打理，相对离车站远，10年，20年都必须要自己修缮，随着年份的增长，房子 不再值钱，只有地值钱，不好卖

日本的房产中介

  日本的房屋买卖市场很规范，想买房子必须通过房地产中介公司。服务确实到位，一个电话就屁颠屁颠开车带你看房子去，还是那句老话，羊毛出在羊身上，这么好的服务背后，是我们要付给他们的报酬，一般来讲，房屋成交价格的3%是中介费，一般买个三千多万的房子，大概就是一百万日元的中介费（人民币6万块），包括了带你兜风带你看房子，想怎么看就怎么看，您说去哪看就去哪看，帮助进如房子内部参观，协助申请贷款，商量事的时候，饮料小吃管够等等吧。

自己盖房子自己卖

  有些建筑公司很有实力，他们不想把这个中介的钱给别人赚，就自己建立一个下属的不动产公司，专门卖自己家盖的房子，因为是自己人，所以这3%的中介费对购房者来说是不可见的，也许他们内部有一定的机制来处理这3%的费用吧

贷款

  相对于中国5%左右的贷款利率，日本的0.625%有点不值一提。对于没有永驻权的外国人来说，只有两个银行愿意为外国人办理贷款，一个是三井住友，另外一个是三菱東京 UFJ，其中三井住友对于外国人的热情更高一些，我拿两个房产申请了两次贷款，都是很简单的就通过了，其实他们评估的基础就是工作稳定性和身体健康状况，如果工作的地方是个超过1000人的大公司，应该很容易就通过。顺便提一下，日本人可以0元买房不需要首付。外国人要首付20%才可以。

意外的知识

  如果贷款者在贷款期间出现了意外身亡，那么他的家属和孩子就可以不用继续背负房屋贷款了，直接获得这所房屋。这还真挺人性的。这也是贷款的时候为什么关注借款者的身体状况的原因。

东瀛故事

发表于 2018-10-22 分类于 印象
  **很早就有想法把发生在日本的事情记录下来，常年拖延症的我总是给自己找借口，这一次，要在自己的站点留下一些东西** ### 日本人的英语

  我工作在一个多元文化的日本公司，有中国人，瑞典人，印度人，尼泊尔人，法国人等等。当然95%还是日本人。这么多国家的人一起工作，工作语言必须是英语，日本人的英语水平差相信大家都有所耳闻。这不能怪他们，因为日语中充斥着大量的英语直接导入使用，他们的做法是把导入的英语找到匹配的日文发音，然后大家都这么念。这样做的后果就是他们听到真正的英文发音的时候就不明白了。
  还有就是有些发音在五十音里是不存在的，比如V，如果发音：Vehicle 他们会说：Behicle， 还有就是他们发儿化音有很大的问题。既然如此，当有些事情无法被改变的时候，我们自己就需要稍微做出一些变化。比如：当你跟日本人用英语会话的时候，你应该通过简单对话很快知道他的水平，如果她的水平不是很好。你要适当的调整自己的语速，尽量用简单的单词，少用定语从句，用最简单的，最有效的方式提问和回答。
举一个反面例子：

在一个skype会议里，一个印度人用速度极快且带Hindi口音的英语跟一个日本人讨论问题，显而易见，那个日本人蒙圈了。因为他连正常语速的长句子都不能完全领会，他就让印度人再说一遍，再说一遍，再说一遍。结果那个印度人下一遍为了简短总结，用了更长的定语从句和更快的语速，结果，这次是真懵逼了！我只好接过话题，把印度人的问题拆分成几个小问题，一点一点的问日本人，这才结束了这场耽误大家时间的对话。会后日本人对我表示了感谢，并道歉说自己的英语不好影响了会议的节奏。我说你不必介意，英语都不是我们的母语，只是我们交流的一种工具，把事情解决才是目的，并不是比拼谁的英语好。当然了，印度人还在跟我抱怨日本人的英语。 关于这个话题还有很多故事可以说，这次就先开个题，

你若盛开，蝴蝶自来

发表于 2018-02-19 更新于 2018-02-20 分类于 生活

  我们生命中的一切所愿，其实不应该用“追求”，而应该用“吸引”，佛说：“有求皆苦，无求乃乐”

  曾经，有一个人为了得到美丽的蝴蝶，边买来一双跑鞋，一只网子，穿上运动服，追逐奔跑了很久，终于在气喘吁吁，满头大汗中抓到了几只，可是蝴蝶在网子里恐惧挣扎，丝毫没有美丽可言，一有机会，蝴蝶就会飞走，这叫“追求”

  另外一个人也很喜欢蝴蝶，于是他买来几盆鲜花放在窗台，然后静静地坐在沙发上品茗，欣赏蝴蝶翩翩而来，心情犹如吸蜜的蝴蝶，这叫“吸引”

  “追求，是从自我的角度考虑，忽视了事物内在的微妙规律，所以常常事与愿违。“吸引”则是从完善自我，奉献自我出发，顺应了天理，投其所好，因而皆大欢喜。

  “你若盛开，蝴蝶自来”；你若精彩，天自安排”

你没有变强是因为你一直很舒服

发表于 2017-10-31 分类于 生活

摘抄自微博，誊写下来，督促自己

  这个世界上有两种人，一种人是强者，一种人是弱者。强者给自己找不适，弱者给自己找舒适。想要变得更强，就必须学会强者的必备技能，那就是让不适变得舒适
  如果你学会了这种技能，你就可以搞定很多事情，例如克服拖延，健身，学习新语言，探索未知领域等等。但是很多人都倾向于回避这种不舒适，毕竟没有一件事是简单的，都需要付出很多努力，忍受很多痛苦，甚至是让自己遍体鳞伤。例如，你想要改掉拖延的毛病，你必须要在任务出现的时候给自己设定时间期限，然后立刻去做，直到任务完成。但是在做的过程中会有各种各样的干扰，你必须要与之相斗争，其过程必然是痛苦的，这种不适应会持续很久，直到有一天你习以为常了，那么你也就不再拖延了。这个过程，就是让不适变得舒适，变得像睡觉刷牙一样，成为一种习惯。
  我以前一直很觉得我们应该让自己舒适一些，但是后来我明白一些不适有时并不是坏事。事实上，你可以学会享受这些不适，例如，我每天都会做一些力量训练，虽然这点不适不会严重到我讨厌的地步，但是人就是这样的，能逃避的困难，我们总能找到借口。我开始告诉自己，既然锻炼肌肉这点痛苦和不适并不影响什么，我不如坚持下来，看看结果怎么样。于是，我制定了计划表格，让这点不适参与我的生活，形成一种习惯。每天我完成15个引体向上，我会在引体向上那一栏写上15，每个月我都会换新的纸张，并总结上个月的情况。不经意间，几个月时间我已经做了1000个引体向上了。
  用同样的方法，我们可以训练自己早期的习惯，其实每天闹铃一响就起床和再睡几分钟起床效果没有什么区别，但是，前者会有点小不适，后者会更舒适一些。我给自己制定了闹铃一响就起床的计划。只要我能在闹铃响后立刻爬起来，就在纸上写下+1，如今，已经坚持超过200+了。
  后来，我发现任何只要是有一点不适的事情都是可以训练的，我们可以将一件不适的事情变成一种习惯，然后你会离不开它，觉得这点小痛苦其实是平淡无奇生活中的一种调味料。这件事由不适变得舒适，良好的习惯就是这样养成的。
