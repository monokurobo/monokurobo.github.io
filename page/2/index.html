要爆发就痛快点

发表于 2020-04-01 分类于 碎碎念

四月一号 - 静待爆发

  相信世界人民都知道日本的疫情爆发只是一个时间问题，但是随着今日东京感染确认人数的不断刷新纪录，日本人才真正的开始认真面对这个事情，到底是不是要封锁东京，这个决定是很难下达的，估计不到万不得已，日本是不会封城的。毕竟这个太伤筋动骨，对于经济的影响不是个小数字。但是对于普通百姓来说真的是很难啊，没有国家的命令，大家还是要该上班就上班，没有理由缺勤啊，但是通勤吧，各种危险，电车、同事、工作环境，无论哪个点中招了，都是要被感染的。现在的心情很复杂，当然了，还是希望日本不要爆发，但是现实客观的数据告诉我们是肯定会爆发的，我现在唯一的诉求就是：要爆发也可以，那就快点爆发，然后难熬个6-10周就过去了，但是如果继续这样掩耳盗铃的通过降低检测数量来达到感染数量的不急增，实在是有点担心！
  今天是愚人节，相信没有多少人有心情来搞恶作剧了！但是还真有一个韩国小鲜肉宣布自己得了新冠，然后又解释说自己是开玩笑的，我操，这种货色直接就该扔到传染病医院让他遂愿！

四月二号 - 在家办公

  不敢去公司了，昨天又确诊了200+的人，这还是确诊人数，没有确诊的那些人还在每天的若无其事的上班，真不知道什么时候日本才封城

四月六号 - 继续在家办公

  上午安倍发表了“发表紧急事态的决心和准备”，估计明天7号才可以宣布，真行啊，日本节奏的玩意太多了。这个时候就体现了共产党的一党专政的好处，一声令下，全国统一行动，快速执行，这才是处理紧急事态该有的态度，感觉日本的东京就是下一个纽约。现在貌似日本的佛系抗疫真的不行啊！但是仔细看了一下那个“紧急宣言”，发现那个东西根本没有强制约束力，该上班还得上班，想出去还是能出去，我操，这和现在有什么区别呢？
  阿比今天上午去OK mart准备储备一些粮食，结果到了门口又回来了，人太多了，他不敢进去。

四月八号 - 紧急事态宣言

  昨天宣布，今天凌晨起正式有效。小学入学式取消，但是还是要去学校拍拍照，领书啥的。
  Stella Town关闭，除了7-11超市，Bic carema关闭

四月十三号 - 强制在家休息

  上周五公司的townhall会议确定了未来3个月的抗疫策略，那就是强制在家休息，每个月8天，这些天里的工资降薪到90%，其他的时间还是在家办公，全额薪资。对我来说挺好的，本来项目也弄的差不多都停了，也没什么事情可以做。
  为了响应安倍的“紧急宣言”，绒绒的小学也是开学就放假1个月，还是暂时的，随着事态的发展，也会相应变化。一家人都在家，天天好不热闹！
  

如履薄冰

发表于 2020-03-11 分类于 碎碎念

三月十一号 - 病毒在别的地方爆发着

  从最近这些日子的数字来看，意大利和韩国争先恐后的挣第二名，意大利明显速度惊人，而且开始了封城甚至封国。但是这些措施更加促使大家迅速扩散到别的欧洲国家。反作用力惊人。跟武汉封城有一拼。因为大家都怕自己的城市或者国家的医疗资源遭到挤兑并崩溃，到时候就会出现国家无能为力的状态，进而只能隔离等死。我很能体会那些逃亡者的心情。
  回头在来看日本，还是在科学的进行着防疫，但是随着越来越多的检测，越来越多的人被确诊，其实后果跟韩国和意大利没什么两样的，前期你不检测的后果就是病毒携带者天天挤电车，坐公交，跟孩子亲密接触，然后孩子在学校传给别的同学，那些同学再传给他们的家人，然后家人继续坐电车，如此循环。爆发只是早晚的问题，取决于他们检测的速度而已。
  我从3月2号开始改为开车上下班，就是为了避开使用电车。而且使我的通勤时间变短了很多，每天6点左右就可以到家，到家就可以吃饭的感觉真的很不错。从姥姥到日本，我们两个大人和两个孩子都或多或少的有一些小问题，咳嗽，流鼻涕，各种小难受，甚至开始怀疑我们全家都已经中招了，这些只是早期症状。姥姥来了之后，我们家里改变了很多，注意饮食，睡眠，减少外出等等之后。至少是目前为止，我们全家都是健康的状态，对亏了姥姥的神助攻啊！继续保持是关键。
  今儿检查“确定申告”的状态，竟然发现银行账户信息是错的，赶紧找Mimura-san帮忙给打电话修正了。

三月十二号 - 继续上班

  收到幼儿园邮件，为了防止疫情，毕业式不让家长参加了。我跟杨晨挺高兴!

三月十三号 - 越来越担心

  全球股市重伤，疫情波及美国，欧洲，中东，全世界都没能幸免，现在开始，担忧越来越重。

三月十六号 - 绒绒去幼儿园

  今早送绒绒去幼儿园，已经好久没有去了，她倒是充满了期待，好久没有说日语了，还有那么多小伙伴久しぶり，一定很多的话要说吧，人类存在的意义也许就是牵绊吧，从小到老，如果没有一些世俗认为的哪些阶段必须要做的事情，那也是挺无聊的
  4楼的自动贩卖机开始支持CokeOnPay，以后就开始告别硬币了，慢慢的，光带着一个手机出行在日本也变得可能，虽然落后于祖国很久，但是毕竟赶上来了。目前在公司附近的消费，一个手机都可以搞定了。

三月十八号 - 幼儿园毕业了

  昨天是绒绒幼儿园的最后一天，因为疫情的缘故，没有让家长参加。其实真的想见证一下日本毕业式的形式。听说是很感动的场面，这次无缘，等小学毕业吧。

如果再把line japan收入囊中，我觉得他就无敌了。从现在来看，他的前期无限撒钱模式是完全值得的！
  毕竟我是乐天重度用户，回头看乐天，现在刚开始加入移动运营商的行列，推出了在指定地区数据流量无限的策略。但是你的信号覆盖真的是少的可怜，我想无限，但是你覆盖不到啊！那个乐天pay在刚出来的时候我对他抱了很大的期望，但是他除了可以消化我的期间限定积分，似乎没有什么其他的了。也不撒钱，现在au都开始每周撒10亿，我都没见过乐天有类似活动，难道在等着憋大招？等其他公司撒完了再撒吗？

寒冬将至

发表于 2019-12-02 更新于 2019-12-10 分类于 碎碎念

十二月二号 - 正式进入12月

  这个月是2019年的最后一个月，一句废话。
  每次到这个时候，就到了回顾本年，展望明年的时候，电视网络上也开始了年终评选，现在网络节目的影响力对于老年人可能没有什么，但是年轻人看电视的不是很多了。每年的12月份，我也会回顾本年的计划并制定下一年的计划。
  2019年的个人计划一共8个，实现了4个，未完成4个，总体来说是不合格的。下面是逐个分析：
  ”瘦到80公斤以下并且保持到2020”,计划的就很扯淡并且不具有可测量性。下次要吸取教训！比如向santosh学习：跑步100KM，这个就很好评估，只需要把自己的跑步里程数加起来就可以评估。
  ”把天津的房子卖了”，这个计划他没有考量到市场的波动问题，比如说：如果高于多少钱就出手，市场行市不好，肯定不能贱卖对吧。下次要吸取教训！
  ”台湾旅游”,这个是我计划清单里的常客，但是每次都无法实现。首先是财务问题不支持，再者是大陆政府对台湾的旅游政策波动。所以推迟到以后吧，好饭不怕晚
  ”抚养亲族添加老人”, 这个还是主要因为自己的日语不自信上，怕麻烦，所以迟迟没有行动，但是我已经添加了抚养家属到2020年的计划里，明年必须搞一下

十二月四号 - Bonus拿到，这一年就结束了

  每年的7，12月是最开心的两个月份，商家也都是在这两个月份打折促销力度最大，因为他们直到这是每年发奖金的时候，所以各种打折，降价，买一送一，积分返金啥的活动搞得真是不要太热闹。作为来了5年得我们家来说，一切都开始看得很淡，除了老婆还有点蠢蠢欲动，总想着出去浪一下，哪怕是浪一小下也可以。但是在日本这个社会，无论怎么浪，都是要有经济来支持的。
  其实看到奖金的时候，我也开始盘算着可以给自己买点什么东西，但是每每想起，又作罢，总觉得应该省点钱来还债。我有时候在想，等我年纪更大一些，会不会后悔我现在因为经济的些许紧张而不停的挤压自己的欲望。等到了有能力购买的时候，自己已经没有了兴趣呢? 越来越真切的感觉到，人的兴趣真的会变，以前很爱很爱的兴趣都会被慢慢抹平，时间越久，磨的越没有痕迹，慢慢的，就不爱了。

十二月五号 - 口说无凭，留下证据

  华为251的事情闹得沸沸扬扬，我就是大概了解了一下，通过这一起劳资纠纷案，我学到了一个很重要的道理：重要的事情都要留下证据！人类的记忆很短暂，如果不是刻骨铭心的事情，很快就会忘记。其实忘记这个技能是我们人类保命的法宝，如果没有这个技能，我们很多的痛苦过往就会不断涌上心头。但是重要的事情真的必须要留下证据。如果不是这个员工聪明，时刻录音，而且录音证据并且分开保存，他极其有可能在牢狱里度过后半生了。
  今早也发生了一件事情就是，生产环境遇到问题，有一个参数无法在系统中看到，Suda就开始问我QA环境测试的时候是什么情况，还好我保留了发送给BA请求验证的邮件证据，不然这个锅我就要来背了。 其实这两个事情很像，如果当事人拿不出证据，那么就会很快的陷入被动。这是今天的收获。谨记！
  昨晚的毛毛很难哄睡，特此记录，总是找妈妈。身体已经困得不行了，但是就是强撑着不睡觉。
  仔细听石进的《夜的钢琴曲31》，并且Google了他的生平简介-大学学计算机，现在是国企电子工程师，因为看周杰伦在演唱会弹钢琴，所以下决心学习钢琴，在无数个夜里，谱写了共31首旋律。《非诚勿扰2》选择了《夜的钢琴曲5》，让他一下名声大噪。现在开始从第一曲开始听， 突然有种想学习钢琴的冲动，就像他曾经看周杰伦弹琴而萌生学习钢琴的想法，现在的我听着他的曲调，开始幻想有一天，我能够弹出他的这些《夜的钢琴曲31》，该有多好，我相信一切都不晚。

十一月一号 - 年底了！
