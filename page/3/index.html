寒冬将至

发表于 2019-12-02 更新于 2019-12-10 分类于 碎碎念

十二月二号 - 正式进入12月

  这个月是2019年的最后一个月，一句废话。
  每次到这个时候，就到了回顾本年，展望明年的时候，电视网络上也开始了年终评选，现在网络节目的影响力对于老年人可能没有什么，但是年轻人看电视的不是很多了。每年的12月份，我也会回顾本年的计划并制定下一年的计划。
  2019年的个人计划一共8个，实现了4个，未完成4个，总体来说是不合格的。下面是逐个分析：
  ”瘦到80公斤以下并且保持到2020”,计划的就很扯淡并且不具有可测量性。下次要吸取教训！比如向santosh学习：跑步100KM，这个就很好评估，只需要把自己的跑步里程数加起来就可以评估。
  ”把天津的房子卖了”，这个计划他没有考量到市场的波动问题，比如说：如果高于多少钱就出手，市场行市不好，肯定不能贱卖对吧。下次要吸取教训！
  ”台湾旅游”,这个是我计划清单里的常客，但是每次都无法实现。首先是财务问题不支持，再者是大陆政府对台湾的旅游政策波动。所以推迟到以后吧，好饭不怕晚
  ”抚养亲族添加老人”, 这个还是主要因为自己的日语不自信上，怕麻烦，所以迟迟没有行动，但是我已经添加了抚养家属到2020年的计划里，明年必须搞一下

十二月四号 - Bonus拿到，这一年就结束了

  每年的7，12月是最开心的两个月份，商家也都是在这两个月份打折促销力度最大，因为他们直到这是每年发奖金的时候，所以各种打折，降价，买一送一，积分返金啥的活动搞得真是不要太热闹。作为来了5年得我们家来说，一切都开始看得很淡，除了老婆还有点蠢蠢欲动，总想着出去浪一下，哪怕是浪一小下也可以。但是在日本这个社会，无论怎么浪，都是要有经济来支持的。
  其实看到奖金的时候，我也开始盘算着可以给自己买点什么东西，但是每每想起，又作罢，总觉得应该省点钱来还债。我有时候在想，等我年纪更大一些，会不会后悔我现在因为经济的些许紧张而不停的挤压自己的欲望。等到了有能力购买的时候，自己已经没有了兴趣呢? 越来越真切的感觉到，人的兴趣真的会变，以前很爱很爱的兴趣都会被慢慢抹平，时间越久，磨的越没有痕迹，慢慢的，就不爱了。

十二月五号 - 口说无凭，留下证据

  华为251的事情闹得沸沸扬扬，我就是大概了解了一下，通过这一起劳资纠纷案，我学到了一个很重要的道理：重要的事情都要留下证据！人类的记忆很短暂，如果不是刻骨铭心的事情，很快就会忘记。其实忘记这个技能是我们人类保命的法宝，如果没有这个技能，我们很多的痛苦过往就会不断涌上心头。但是重要的事情真的必须要留下证据。如果不是这个员工聪明，时刻录音，而且录音证据并且分开保存，他极其有可能在牢狱里度过后半生了。
  今早也发生了一件事情就是，生产环境遇到问题，有一个参数无法在系统中看到，Suda就开始问我QA环境测试的时候是什么情况，还好我保留了发送给BA请求验证的邮件证据，不然这个锅我就要来背了。 其实这两个事情很像，如果当事人拿不出证据，那么就会很快的陷入被动。这是今天的收获。谨记！
  昨晚的毛毛很难哄睡，特此记录，总是找妈妈。身体已经困得不行了，但是就是强撑着不睡觉。
  仔细听石进的《夜的钢琴曲31》，并且Google了他的生平简介-大学学计算机，现在是国企电子工程师，因为看周杰伦在演唱会弹钢琴，所以下决心学习钢琴，在无数个夜里，谱写了共31首旋律。《非诚勿扰2》选择了《夜的钢琴曲5》，让他一下名声大噪。现在开始从第一曲开始听， 突然有种想学习钢琴的冲动，就像他曾经看周杰伦弹琴而萌生学习钢琴的想法，现在的我听着他的曲调，开始幻想有一天，我能够弹出他的这些《夜的钢琴曲31》，该有多好，我相信一切都不晚。

十一月一号 - 年底了！

  日历又翻篇了，十一月开始了，距离姥姥回去还有11天。在家里贴上了倒计时的牌子，来提醒大家，姥姥在的美好日子将暂时的一去不复返了。
  十月份支出严重赤字，主要是给绒绒买书包和轻井泽旅游导致的。11月份需要合理化支出。
  暴雪嘉年华明天就要开始了，还是很期待暗黑4的。

十一月五号 - 我又来上班了

  三天的小长假很快就过去了，这个星期的重点是为年末调整准备loan的资料
  今天突然又开始想着买游戏笔记本拯救者Y7000P，9299的价格确实有些高，倒不是没有钱，主要是现在家里财政状况正堪忧，怎么可能同意我换笔记本呢？我又没什么刚性需求，就是想玩一些游戏。而昨天我还答应人家，等姥姥走了，我看着两个孩子，让老婆去健身，合着我就会吹！！？算了，压制一下自己内心得欲望吧！

十一月六号 - 突然腰疼的不行

  本来带着跑步的装备来公司，但是路上突然就腰疼的不行，撑着完成了当天的工作，下了班直接去了家门口的接骨院，运气很好，直接就接受了治疗，医生这次重点按压了我的双腿病开了跟上次一样的膏药，说肌肉会至少疼三天。有啥别有病，真不是开玩笑的。
  今天腰痛好了很多，可以站立办公了。外面阳光灿烂，真想出去晒晒太阳。那就去！
  前些日子，我向往着Amazon Echo，跑步后想买无线蓝牙耳机，可是当这些小心愿都很快满足后，我并没有很很经常的持续使用他们，就是新鲜劲儿过去了，我在想，如果买了电脑之后是不是也会这样。我对什么都提不起兴趣了。没有新的兴趣，老的兴趣也在渐渐消亡，难道这就是长大的感觉吗？还是我追求的不对，我需要更加的完善自己，充实自己，挣更多的钱，不是不可以，但是我想问：然后呢？？也许有人会觉得我是“无病呻吟”，但是追求的空虚感真的越来越强，不知该怎么样。

美丽的秋天

发表于 2019-10-01 分类于 碎碎念

十月一号 - 新的月份开始了！

  早晨时候就开始查看绒绒幼儿园的园费收取情况，就想看看政府到底给减免了多少，结果还是蛮喜人的:减免了2.5万左右，如果按照增长2%的消费税来看，我要消费125万的日币，才会是2.5万日币。由此可见，日本政府的执行力度还是很给力的。我们其实是这个项目的实际受益人。绒绒可以用几个月，毛毛倒是可以从头适用。如果每个月可以减少2.5万的支出的话，对于我们的生活质量还是会有帮助的。
  今天是国庆，二姐出游，大姐夫加班，只有哥一个人去爸妈那里过节，妈还说猪肉已经涨到22块钱一斤了。呵呵！
  从10-12月份，绒绒的校车改成晚bus，不需要我送了。顺势开始了我的跑步健身之路。带着我新到的无线耳机，7点40开始跑步，30分钟3.6公里，跑完之后整个人都精神了，很爽！见面打招呼就可以感觉的出来。怪不得日本人见面都会问：”お元気ですか?” 看来人的元气/精神头真的很重要，今天还搜索了一下成人每天的卡路里摄入量。争取控制在1800-2000，这样坚持下来的话，身体的整体状态是会变好的，减肥不减肥倒不是很重要，还是要看BMI和体脂率。
  Hexo的日志按照更新日期排序的问题解决了，告别了手动的方式，开心一下。参见这里

十月二号 - Package Syncing

  随着设备的增多，软件的配置和偏好同步问题逐渐成为了问题，比如自己常用的SublimeText，目前还只有2台工作电脑的我就很苦恼他们之间的同步问题，有的时候在公司的电脑配置好了，回家以后又不能如预期的工作了，今天参照这里完成了Sublime和Google Drive的同步。这样就可以在任何机器上快速得到自己常用的package然后开始工作了。
  今早突然告诉绒绒我要带她出去玩，而且每个月都有一次机会，她虽然极力克制自己的高兴，但是在我出门的时候，她还是跑过来，异于寻常的跟我告别，从这里我知道了，我不是一个好爸爸，跟她互动的太少了。现在我要把这件事写到每月必做的todo list。

十月十号 - 看病归来

  入院->治疗->出院，三天的流程把我们折腾的够呛，但是无论如何，总算是安全无虞的结束了，一切都是好的结果。中间还穿插了绒绒的小学入学体检。发现她的眼睛有点异常，需要进一步诊断，其他一切都顺利。毛毛回家后就开始爬，玩，跟没有任何事情发生一样，让我们对日本的医疗技术有了全新的认识，而且整个手术一分钱都没有花，全部由健康保险负担了。真的是很省心。

九月二号 - 音乐流媒体设定

  前几天突然，真的是很突然就购买了Echo Dot第三代，主要原因是Amazon又搞活动，因为他老是给我推送邮件，说Echo又便宜了，这一次我败了，被他们这轮推送给彻底打败了，我早晨起来上厕所，就在amazon上一路点， 直接入手了。等货物送到的时候，老婆还责问我，倒不是因为钱的事情，她在意的是我没有跟她商量就买入了，下次不可以这样。谨记，很多时候真的不是因为钱，而是因为家庭里的相互尊重。
  买了Echo，下一步就是要选择音乐流媒体提供商了，主要是Apple Music，Spotify，Amazon Music。虽然现在Echo已经开始支持这3家，但是各有利弊。因为Echo只支持日语，所以即便Spotify和Apple Music的中文歌服务更好，也没有办法用日语点歌播放中文歌，但是因为spotify可以免费3个月，所以先用着看看，spotify和Echo组合有什么效果再说。

九月四号 - Echo控制空调成功

  用了一个晚上的时间完成了Daikin空调跟Echo dot的互动，现在不但可以远程控制家里的空调，而且可以用声音对Echo下命令来控制空调了，用姥姥的话说，这么贵的空调也没有白买。日本真的是一分价钱一分货，但是我们搬家的时候购入的空调，冰箱都挺合适的，冰箱只买合适的，没有买乱七八糟黑科技的，而空调呢，我们选择了节能，智能的Daikin空调。不错的选择
  老婆想报JLPT的日语考试并且想直接N2起步，我总觉得有点担心，她虽然很聪明，但是学习时间还是要有的，现在她在照顾孩子之余，看手机的时间都没有，更何况去踏实去学习呢。所以我建议她N3起步。

如火的八月

发表于 2019-08-27 分类于 碎碎念

八月五号 - 没别的，就是热！

  一晃就到了八月份，日本的八月真的是不适合居住，热还罢了，关键是湿气太重，出门就是一身汗，更别说在人多又拥挤的月台了，
每次到了这个月份都是坐电车上班族的噩梦，比不了那些开车上班的人，人家车里有空调。电车里面是一点也不热的，但是在等车的时候，那个汗水啊，顺着裤管往下流，内裤都湿了！无论男女，手巾擦，扇子扇，除汉湿巾等等，今年较往年还好些，我搬家到了车站的附近，走两分钟就可以上电车，可以找准时间去赶电车，然后换乘一次就可以到公司了，比以前强了太多，以前要走10+分钟才可以到车站，到了就热的不行了。
  周五邮寄给泽泽的Switch机器还以为这周一就可以到了，谁知道邮局打电话说电池有问题，需要分开邮寄，我的天，邮寄那天怎么不说清楚呢！
不过还好，妈妈刚好要去邮寄奶粉，要不然真是要白跑一趟。我查询了网上才知道原来Switch里有3块电池，而EMS的规定是只允许有两块，所以只能把手柄分开邮寄。这么热的天，妈妈带着两个孩子还要邮寄，真的是辛苦她了。不过还好，姥姥就要来了，可以让老婆好好的休息一下了。
  今天早晨称体重，竟然90+Kg了！！！这还了得，我今天的计划体重可是80-Kg。必须要行动起来了，今天中午跟Santosh商量了一下，决定中午跟他去公司的健身房锻炼，20-30分钟。今早在电车里看到了一个大概40岁+的一个女性，还在拿着英语单词书在背单词，在那一瞬间，我惭愧了，真的是惭愧的不行，人家那么大岁数都还在奋斗，而我却在自己的安乐窝里天天打游戏度日。我要改变，从今天开始！

八月六号 - 听听老歌

  最近在使用Amazon music，速度，体验都还不错。就是中文歌曲不是很好找到。无所谓了，瞎听呗。周杰伦的歌曲还是可以找到的，现在听着他的《开不了口》还是别有一番风味的。

炎热开始的七月

发表于 2019-07-24 分类于 碎碎念

七月二号 - 半年已过

  2019年过半，回头看看达成的”成就”,似乎没有。从1月26号搬进新家，就开始忙活，孩子的身份证，护照，Mynumber，役所的各种手续。目前来说，一切都已经上了正轨。也不能说什么也没有做，但是这些都是应该做的。
  一切都顺调之后，遇到了一个问题，不知道该往哪个方向努力了。最近的一年，对IT技术已经生疏了很多，更多的是关注测试方面的计划，统筹等等。随着跟各个部门打交道，自己的心性和交互能力提高了很多，知道了什么可为，什么可以变通，什么可以拒绝。但是隐隐的有些担心，如果明天我被通知不要来上班了，我可以找到别的工作吗？
  人的发展有很多中可能，为什么我选择了测试岗位？是觉得技术开发已经没有什么意思了。我想继续转管理的岗位，如果问为什么，首要的是薪水会提高，然后才是其他的原因。没有必要不说出心里的真实欲望，我也知道，管理的岗位有管理岗位的苦处，不能光看贼吃肉，不看贼挨打。

六月三号 - 继续工作

  周末的肠胃病毒发作，大部分时间都是在厕所里度过的，肚子都被拉空了。强撑着来公司上班，中午就在Lawson买了一份面条，不敢多吃，因为吃了的东西马上就会排出去。我是怕了！
  这已经是很恶心的事情，但是吃饭的时候听到了更加恶心和荒谬的消息，石头居然要当架构师了，我的天。他当个开发人员都不够格，屁都不会的一个人，居然去当架构师，真的是亮瞎了我的氪金眼。最关键的是，领导们居然也同意了，就由这他这么瞎搞吗？当时想着，这他妈的太扯了，如果他能胜任，我觉得便利店的大爷也可以的。一点相关培训都没有参加过，就去当架构师了，UD的人士任免真的是玩笑。其实这本来就不关我的事情，人家爱干什么干什么，又没有影响我的收入。

六月七号 - 难得的周五

  紧张的4天过去了，每天都跟打仗一样。今天抽时间搞清楚了投资信托，原来我之前照猫画虎的乐天薅羊毛不是买的股票，是买的投资信托，而且必须是積立注文才可以使用信用卡支付。第一次的支付和赎回已经结束，赚了680，算是第一次尝试吧。第二次估计要等月底再卖出吧，本来就是想薅羊毛赚取一些积分，没想到资本市场还是很好玩的，以此为入口可以继续理财生活。
  最近很多的大龄明星公布婚讯，大家都觉得不般配，说好白菜被猪拱了，看了日本那一对，男方是搞笑艺人，在新闻发布会时，有记者故意刁难说他爱人是心机女，他不卑不亢，分分钟霸气控场，最后一句话还能把快哭的女方哄回来，这份淡定从容，情商爆棚的男人，如果我是女的，也会义无反顾的嫁了。年轻的时候看颜值是对的，因为荷尔蒙作祟，都会挑选自己看着舒服的，这无可厚非。但是三四十的时候，就渐渐明白了，能找到一个三观合适，有话说，呆在一起舒服的人长久的过日子是多么难。吃瓜群众真的够累的，人家结婚，哪容得你们这些妖怪反对！
