十月一号 - 十一月正式开始

秋风凉

发表于 2020-10-01 更新于 2020-10-27 分类于 碎碎念

十月一号 - 十月正式开始

  9月份就这么过去了，十月开始渐渐变凉，早晨要穿长袖的衣服。
  今早绒绒忘记了戴帽子，我回来找也没有找到。这是第二次了，第一次是没有带联络簿，让我那么一顿跑才追上给她送去。这次连帽子都没有带，然后妈妈还跟姥姥视频的时候抱怨说：哪有孩子不忘事情的！？这是什么他妈的理论？我从来没有怪绒绒，明明就是家长的责任，她也说了，这是她的疏忽，但是听不出一丝不好意思。对自己可真好！让其他孩子和家长怎么想我们：她们一家都是马大哈？天天忘东忘西？我是没有办法再经理第三次了，我在门口贴了纸，提醒每天必须要带什么，让绒绒在出门前仔细核对。

十月二号 - 无题

  姐姐突然要跟我聊聊，我是真的没有想到。但是她给我讲的东西我听进去了。我耍脾气的處理方式确实不对而且不明智，只会增加解决问题的难度。还是和好吧。

十月五号 - 累的毫无价值

  白天在班上工作，下了班就直接回家，想着可以帮着她做做家务，带带孩子，真的是累成狗。十月三号，四号这两天，我也是尽我的全力去服务家庭，那种累，已经让我的身体有了反应。但所有的付出没有得到任何认可，反而她在跟她妈视频的时候狠倒苦水：“她要累死了，压抑死了！！！”我艹，我真的贱。原本我以为，我有着一个美满的家庭，一儿一女。慢慢的，我感觉，这种表象的背后，有一个人的欲望永远不知足，每个人都会要更多，更多！在既有顾家老公的基础上，她想要让他在不是工作的时间干更多的事情，分更多的家务，带孩子更多时间。我完全感觉不到这个家庭对我的爱，对爸爸的尊敬，这个家庭只会慢慢的变的畸形，我现在惧怕回家，一回家收到的都是负面的反馈，收到的都是“我这个还做的不好，那个还需要改进等等”。我累了，我需要改变。从2020-10-05这一天开始，我决定不要再做一个顾家的爸爸。

十月六号 - 鬼使神差

  自己挣扎了半天，但是昨天回家后看到孩子们的笑脸，白天想到的种种都立刻烟消云散，继续做起了好爸爸的角色。

十月七号 - 十点才下班

  本来打算在公司解决的，但是因为某些原因导致了回家以后开会开到晚上10点。头昏脑胀,然后再暗黑的世界里疯狂的割草，发泄掉一天的疲惫。

十月八号 - 网易云音乐

  随着版权意识的迅速提高，在国外听国内音乐很麻烦，必须要翻墙回国才可以，Goolge了半天，终于被我找到了一个解决的方案：
网易云音乐+UU加速器，UU加速器是网易自己家出的加速器，只能为网易的应用或者游戏提供翻墙加速等服务，估计很多住在国外但是想吃鸡的玩家都会用这个加速器吧。使用了一下，真香啊，最关键是，如果通过stream收听，这么好的东西竟然是免费的！真的无法相信，真的是业界良心，人家给你软件让你翻墙回国，还可以免费听歌打游戏，真好！
  网易云音乐的APP界面做的也很棒，各种小细节和便利，国语歌曲全部收录，赞！👍
  预约流感疫苗，可以直接在公司接种，还是很方便的
  申请“年末调整”的网上程序，这样就不会接受到纸质的材料到家里了，日本的电子化越来越好。
  也许是因为下雨的原因，突然很想抽烟。已经很久没有抽烟了，靠VISA touch的活动买了一包烟，下雨天抽烟真的很有感觉，这个习惯是在瑞典出差的时候染上的，那个时候刚好赶上了那边几乎极夜的月份，下午2点天就黑了一直下雨，一个人在外，郁闷无法发泄，只好靠抽烟排解。

十月十三号 - 重新出发

  昨天带着家里人一起去打流感疫苗，毛毛连眉毛都没有眨一眨，绒绒却拼死抵抗，真的是拼死啊，用了全身的力气，要是我没有一起去，一般人还真的不能控制住她的身体，太野蛮抗争了！虽然最后成功的打了疫苗，但是哭了好久，最后也承认并没有想象的那么疼，希望第二次的注射可以轻松些。
  可以说昨天是家庭气氛重归于好的第一天，就是绒绒最近几天一直不能好好入睡，总是用各种理由不睡，昨晚妈妈跟她急了，说了她很久才明白原来是她想跟妈妈弟弟一起睡，所以才找各种理由不睡觉，希望可以跟我换，哎！
  我觉得我应该放更多的时间和精力到工作上了，在这个风云变幻的时间里，最好是要找到自己的位置和不可替换的理由，不然说什么都是虚的。

十月十四号 - 丧

  跟姐姐聊了会天，说到了孩子的“丧”文化，对待事物的态度总是消极的，没有追求，觉得目前拥有的就足够好了，不想要更多的任何东西。

十月十六号 - 被迫放弃SublimeText

  今天早晨收到公司的邮件说Sublime对于商业使用不是免费的，所以我们必须要卸载掉并给出了一些替代方案，但是那些都是好傻的备胎，我是不会用的。最后重新拾起了Atom这个完全免费的文本编辑器。稍微试用了一会，觉得跟Sublime很相似，插件丰富，易于上手，没有过一会就通过插件恢复了Sublime的使用环境。有点意外的不错。
  折腾了一整天的Atom，对于Python的IDE-terminal还是没有成功，但是细想也觉得没有那么必要，如果有需要执行的脚本，直接去Mintty里执行就好，没有必要非要在文本编辑器里执行脚本和看结果。

十月二十号 - 一来公司就放松

  也许是最近工作不是很忙的原因，一到公司就感觉特别的轻松，总是不想提升自己，就是摸鱼。心里也是很矛盾，如果天天这样摸鱼不就废了吗！一边担心一别继续摸鱼，艹！
  这几天两个宝儿都得了感冒，打喷嚏流鼻涕不停。做家长的最怕孩子的病，一个得了，另外一个也90%会得，麻烦得很。
  老婆的Iphone手机确实用了很长时间了，需要换一个新的了。关键是最近感觉乐天的活动越来越不行，paypay也是乏善可陈。倒是docomo的各种活动让人眼前一亮，越用越顺手，所以索性跳入docomo的生活圈也是可以的。暂且试试再说。

十月二十二号 - 普通的一天

  昨天从老高那里学习了抑郁症的表现，对照了一下，我是没有抑郁倾向的。但是或多或少的了解了抑郁症的痛苦，对所有事情失去兴趣，对吃的失去兴趣，不停的思考自己是否有活下去的意义，真的是很可怕。

十月二十七号 - 年末又开始忙活

  每次到年末，就到了年末调整，车辆保险更新，财产税缴纳等等，今年选择了“简单年末”就是不接受纸质的申请表，一切都在网上解决。但是今天在录入的时候，首先是亲属证明，送金证明搞定了，但是又蹦出来住宅贷款的证明要求，没有准备，只能明天再弄了。这次要好好的记录下来作为未来的范本。税务署邮寄给我了剩下9年需要用到的证明材料，银行也给我邮寄了剩余贷款明细的证明书，一切都是那么的及时和规范，果然不愧是日本人的风格。一切事情，到了该到的时候，总是会到的！
  今天在lawson使用D Point失败，仔细的查询了他们的网站发现了问题，回答的明细令人发指。这突然让我想起了看到的一个文章说：如果在日本生活的久了，人会变成”傻子”，因为很多事情都是定义好的摆在那里，你只要按照需要的东西提交就可以了，没有腐败，不需要走关系，不需要找人处朋友。一切都是清楚明晰的摆在那里，只要follow就好。但是这也让人和人的距离因为互相没有利用的关系而变得生疏，孰是孰非呢？我更倾向于日本的一切照章办事。

九月二号 - 还是购入了

  思量再三还是入手了，价格不菲啊!肉疼了一会儿，但就一会儿。剩下的就是期待快点送到家。
  昨天交付了Testbot的事情，今天就轻松了很多。

九月七号 - 正式上班

  最近妈妈的大姨妈来了，对绒绒的发火的态度真的是无法控制。很小的一件事情也可以升级为大声的训斥。看来激素这个东西真的是很可怕，控制着人的一言一行。

  给姐姐打电话，姐姐说她不想活得太久，七八十正好。因为最近这3年她在赵宇泽的事情上已经心力交瘁。不想活太久。青春期的孩子真的是让人头疼啊。我姐那么好的脾气都受不了，我跟杨晨会如何呢？

New Normal

发表于 2020-08-03 更新于 2020-08-31 分类于 碎碎念

八月三号 - 绒绒跳舞

  昨天去送绒绒跳舞，被老师说：没有穿橘色的内衬，其实老师很早就在Line里通知了，让大家自行购买，但是我没有认真的读老师的信息。没办法，下周二就要发表会了，再买也来不及邮寄到家了。于是赶紧去Stellar Town搜寻，UNIQUO，GU，RIGHTON，能找的店铺都搜索了，没有发现合适的橘色。到了接孩子下课的时间还是没有找到。但是触动我的一幕发生了，竟然有两个妈妈在一开始听到我没有买到内衬后，她们去UNIRUO帮我找到了合适的颜色并拍了照片。当时的心情特别难以形容，我根本没有拜托人家帮我找，那两个妈妈也是素昧平生，但是她们就默默的帮我找。如果我说我买到了，她们估计就不会告诉我她们帮我找了这件事情。越想越温暖。这个世界如果多一些这样的事情，坏人也会少很多，我坚信，世界上没有纯粹的坏人，她/他们一定被别人恶意的对待过，所以才对这个世界充满了敌意。我无法表达我对那两个妈妈的感激，但是我会传递这份善意到我可以到达的地方。
  这周的重点就是帮姥姥准备好回国用的东西。
  还是在不自觉的看电脑的评测视频，难道我不买就一直这样下去吗？如果我买了这个电脑，我会用它来做什么呢？游戏的话：炉石，魔兽，暗黑3和未来的暗黑4，三国志14。处理视频：家庭视频，旅游视频。还有什么呢？好像也就没有什么了！

八月十七号 - 暑假归来

  长假结束了，我开始上班，绒绒开学了。一切似乎回到了正轨上。离开了电脑这么久，回到公司碰触到键盘的感觉真的挺好的。又一次庆幸，自己的工作跟自己的爱好是合的。
  这次回到公司，收到了很多的土产碎片，还有工会发的除菌湿纸巾。久违的感觉！
  姥姥成功的回国无疑是这个长假最重要的事情，还好一切都很顺利，我开车来回成田机场也是成为了我人生的首次机场接送任务。到达中国后的检测也为阴性。希望姥姥可以在结束隔离14天之后安全的回到天津的家。
  本来还为Test Bot的事情忧心，但是又一次忘记了小兵的法宝-向上反映问题，让领导定夺即可

八月二十一号 - 又是一周

  今天是周五，匆匆的一周很快就过去了，完全记不得自己干了什么。只是在既定的生活节奏中活跃着，完成着自己作为父亲必须要完成的事情。似乎没有任何让自己兴奋的事情发生。一切过的都是那么的自然，那么的无声无息。
  昨天妈打电话过来询问是否可以降价卖出铁成的房子。要降价10万销售，我觉得还是不要着急出手才好，毕竟那个房子是一个可以下蛋的鸡，杀鸡取卵也没有什么好处可言，今年年内如果可以把明伟哥的钱还清，后面就会好很多的。所以还是先保留着吧。再者说中国是没有房产税的。

貌似要恢复如前

发表于 2020-07-01 更新于 2020-07-31 分类于 碎碎念

七月一号 - 2020已过半

  虽然不好的事情一直在发生，但是时间并不会停滞，还是在滚滚向前。眼看着6月份就过去了。日本的疫情虽然看上去控制的不错，但是还是需要倍加小心。随着时间的推移，这种警惕传播的意识还是要时刻保持着。在家上班一定会成为未来的主流，那么问题来了，怎么才能在家里快乐的上班呢？我现在的居家办公环境不是很好，首先是位置，因为信号的原因只能在孩子的活动室，他们在旁边很吵，根本无法沉浸式办公，只能一边忍受一边勉强完成工作任务。所以我比较喜欢来公司上班。
  今天跟老钱聊了一会儿，表达了我对不可知未来的担心，她也例行公事般的开导了我。问题不大，从中也得到了一些消息，她的合同本来是今年7月到期，但是她延期到了明年9月份。而且在日本方便她需要找到一个她这个位置的替代者，目前Suda是候选人，但是也不确定是否找到合适的人空降。

七月三号 - 从来没有过的冲突

  起因很简单，绒绒在饭桌上屡次不改的看着毛毛那个年纪的动画片，我先说她，她没有改，然后姥姥说她，她还是继续，妈妈也说她，她还是看，然后我跟正式的说：“三个大人都说你了，那是1岁孩子的动画片，你都7岁了，别再看了，赶紧吃完饭再说。”可是她还是当耳旁风。当我再次发现她还是看电视不吃饭时候，我无法控制我的怒火，我说“你给我滚。”然后妈妈，姥姥就立刻跳出来开始劝，越劝我越火大，越劝我越更大声的让她滚，她被吓到哭了。但是我还在继续，妈妈和姥姥都觉得我疯了，小题大做有病！而我在那个瞬间就想让绒绒服从我的指令，无论谁都无法改变的指令，其实更多是想证明，爸爸在这个家是算数的。但是到最后，绒绒也没有离开饭桌，我说了一句：“好，你不滚，我滚！”事情并没有结束，到现在，他们两个都认为我过分了。 也许妈妈会理解我一些，她知道我是想证明爸爸的权威。姥姥还在怪我，从平时的举止可以看出来，她总说：“等我回国了，你们再教育她，我在着的一天，就不可以这么对绒绒！“
  我一直很尊重姥姥，她说的任何话我都听，即使有的时候有些微词。关系处的也还算不错。但是在教育孩子这件事情上，我们爆发了从来没有的冲突，有些人可能会说， 你作为一个父亲，应该怎么样怎么样，不应该凶孩子。我会看是谁在说这句话，如果你还没有孩子，我会说：”去你妈的，等你有了孩子再来说这句话！“ 如果他已经有了孩子，我倒是想看看他的孩子教育的是个什么样子。我不是不看育儿的书籍，我不是不想当一个好爸爸。 我认为：任何事情都有度！如果孩子一直在探测父母的底线，这个时候父亲是有责任站出来，给她一个信息：“你这么做，我们很不高兴。”孩子是很聪明的，他们可以感觉得到父母的底线的，我们平时无论怎么爱他们，保护他们，给他们最好的成长环境，资源。但是不能没有底线！一旦越线，就必须严厉的告诉他们。后来冷静下来，我想了想。每个人想问题的出发角度不同：姥姥就想保护绒绒，因为绒绒跟她时间最长，感情最深，她不希望绒绒收到任何的伤害，哪怕是来自她的父母。妈妈在中间也很为难，一边是妈妈，一边是爱人，对象是孩子。
  我还没有想好如何结束这种尴尬的场面

  ポイント的战争还是没有结束，但是已经快完事了。现在的阵营感很强，Yahoo靠Paypay打了一个很漂亮的翻身仗，DOCOMO也是发力很足，AU也在积极应对，唯独乐天还是吃老本，活动少的可怜，还在玩乐天point和Edy，大哥，快醒醒吧，过不了多久，你的point也玩不过其他人了，因为人家已经靠各自的pay来牢牢锁定用户的point，你还在捣鼓你的漫天撒花point！作为一个用户，我只能选择在对的时候，使用对的服务，甚至有可能因为使用某个阵营的服务多了，从而转向使用那个阵营的全部服务，比如电话服务，信用卡，银行，所以现在的这些玩家都是有移动运营商背景的。平心而论，paypay这个app是做的最好的，没有之一。而且随着时间的进行，它会慢慢整合yahoo信用卡，prime用户，yahoo钱包, 拍卖，二手货等，paypay mall也会替换yahoo shopping，我觉得yahoo的战略规划非常的清晰，就是依附paypay这种支付服务，把以前的网页版服务全部变成移动版。一切都向paypay靠拢。
如果再把line japan收入囊中，我觉得他就无敌了。从现在来看，他的前期无限撒钱模式是完全值得的！
  毕竟我是乐天重度用户，回头看乐天，现在刚开始加入移动运营商的行列，推出了在指定地区数据流量无限的策略。但是你的信号覆盖真的是少的可怜，我想无限，但是你覆盖不到啊！那个乐天pay在刚出来的时候我对他抱了很大的期望，但是他除了可以消化我的期间限定积分，似乎没有什么其他的了。也不撒钱，现在au都开始每周撒10亿，我都没见过乐天有类似活动，难道在等着憋大招？等其他公司撒完了再撒吗？

三月二十七号 - 世界不好了

  隔了这么久没有写字了，不知道什么原因，不是因为懒惰，这些日子被各种信息包围着，已经分不清哪些是重要的

  1. Volvo瑞典员工全部阶段性解雇，等疫情过后再召回来。
  2. 日本东京奥运会延期到2021年
  3. 美国股市多次熔断，惨烈可想而知
  4. 割肉卖出比特币
  5. 印度LOCKDOWN
  6. UD开大会，生产可以坚持到第21周
  7. 小何无法转为正社员而选择离职
  8. 美国350万人失业登记
  9. 石油价格暴跌
  10. 通过确定申告拿到退税

  看到上面随便就列出的10条信息，再看看现在的自己，突然感觉好幸福，我现在还有工作，还可以坐在办公室里打字记录心情，那些被疫情困扰的国家和个人，那些丢掉工作的家庭，那些被股市蒸发掉的资产，这么一想，很庆幸自己还有工作。不光是挣钱养家的一个营生，而是一个每天起床之后可以去的地方。我的梦想不大，就是老婆孩子热炕头。

  世界上所有的事情似乎都加快了节奏，领导层需要每天开会讨论明天的计划，我最近开始睡不着觉，担心自己的工作问题，不知道哪一天公司解雇我，对于一个中年男人，一个上有老下有小的我该何去何从？以前不理解“中年危机”，现在有了切身体会，就是怕没有工作，怕辜负周围依赖对你的期许。哎！
