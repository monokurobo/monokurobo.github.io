七月七号 - 雨一直下

  进入七月就开始下雨下个没完没了，对我倒是影响不大，但是对于妈妈晾晒衣服就不好弄了，一波一波的洗，前面的没有干呢后面又等着晾。好歹今天天气还算不错，赶紧的吧。
  新买了一个称，小米的体脂秤，昨天到货后全家人齐上阵测试，妈妈综合得分96分，我才得了26分，简直被比到泥儿里了。她也一直开始自吹自擂的反思自己身体状况为什么那么好，一边各种讽刺挖苦我的26分身体。O(∩_∩)O，蛮有趣的。没有这个体脂秤的时候是没有这一套量化标准的，但是突然被数字化之后，一切就一目了然了。数字改变生活啊！我也顺势立了flag：我要以综合得分达到60分为目标前进。今天早晨挣扎着不想从床上爬起来跑步的时候，我就反问自己，你一个26分的身体有什么脸赖在床上，人家96分的人才有资格继续睡觉。
  最近在twitter上关注了小武爸爸，他发的好多优惠信息总是能让我找到共鸣，好多东西我本来没有想到是缺失的，但是他一条twitter推荐，我就忍不住的买了。比如中国风袜子，汽车的遮阳器，瑜伽垫等，如果他不发，我就根本想不到去买这些东西。而且开始践行他推荐的Point环线建设，虽然有好多东西要申请，但是这个过程中我丝毫感觉不到累，反而充满了好奇，日本是一个point的大国，但是很是分散，大家都生产自己的point系统，但是各自为战，就算是几个头部point企业也是不可以互相兑换的，总是希望用point帮助用户，让他们肥水不流外人田的那样继续消费到自己的经济圈里。我可以理解这种做法，但是让我们消费者就很不好管理这么多的point账户，有效期限等等问题。

六月一号 - 新的月份

  一觉醒来，已经是六月份了，炉石新月开始，各种Pay的活动刷新，工作也要进入新的进度，一切都要重新计划起来了。本月的首要大事就是我的护照更新，现在对待各种行政的事情格外的上心，比如前几天老婆把她的驾照更新了，我就觉得是很好的事情，因为这些事情都是必须要办，越往后托越麻烦的事情，早办完早安心。
  この特別な機会をお見逃しなく（みのがし）　为什么总是记不住发音呢！

六月三号 - First Release

  成功部署并且验证成功。

六月八号 - 涨工资

  知道了数目，聊胜于无吧，但是ボーナス却因为新冠疫情影响降成了4.0！F**K
  昨天成功邮寄了办理护照所有需要的东西，心里的石头稍稍的放下了一点，期待2周之后的结果了。护照这个东西只有在国外的时候才能体现出它的价值，国家的强弱直接影响了这个护照的含金量，目前来说，日本的护照是比较厉害的，在含金量排名中一直名列前茅，但是我们国家随着国力的发展，现在也比较厉害了。
  很意外的收到了SMBC的信用卡审核通过的结果。其实已经申请了2次了吧，这次终于通过了。主要原因是当初太年轻，开卡活动的奖励得到后，直接注销了我原来的SMBC信用卡，导致人家把我当成了投机分子，所以再次申请的时候就遇到了这样的情况，其实我也可以理解。所以这次一定要好好的利用这张卡，不敢再注销了。

六月十一号 - 说出现状

  昨天跟她又一次因为是否给孩子报VIPKID展开了争吵。她不知道家里的经济状况而只知道提出要求，昨天给她透了底并且觉得孩子现在报英语班还为时太早，因为她还没有正确的学习态度，还是满脑子是玩的年纪，如果给她报了也达不到效果还不如等三岁她上了私塾后再系统的学习。老婆接受了我的建议。

六月十六号 - 努力的生活

  最近老婆一直在为自己的6万餐费而努力，也不敢像以前那样随便出去吃了，去超市也是用现金了，她说这样看着现金一点点的变少，会时时刻刻的提醒她这个月的生活费不多了，也是间接的省钱了。我同意，这也说明了为什么无现金社会这么火热的今天还是有很多的现金主义者，尤其是日本的老人，首先他们消费就少，而且一直有着忧患意识，有钱不投资也不花就。存钱以备万一，所以无论如何都不会轻易改变他们的生活习惯的。花钱习惯更是难改。
  今天通过跟Abi学些CMS的原理，终于理清了我的很多困惑，感觉每天都进步一点真的不错，总比整天浑浑噩噩的好。

六月十八号 - 三娃警报解除

  老婆不知道抽什么风，总是感觉自己又怀孕了。更可气的是中国开放了三胎的政策，一切的巧合让我还真的有些害怕，老婆是典型的好了伤疤忘了疼的主，但是我对伺候月子和养育孩子可以有切身体会的，所以我坚决不同意。拉锯了2周左右，昨天终于她的大姨妈及时来了，这才彻底解除了三娃警报。
  昨天给老婆说了，如果她能够成功的降低每月餐费，那么节省下来的钱可以用来旅游，吃好吃的，买玩具等等。但是首先要完成每个月6万的计划。
  从6月15号开始，我开始了16-8小时进食法，就是晚饭之后16个小时不吃饭，然后在8个小时内可以正常的进食，然后继续16个小时不吃，如此往复，今天是进行的第四天，我还在严格遵守，昨天晚上吃饭比较晚，所以今天要到11：30 才可以进食。刚才肚子叫的好凶啊。我忍！
  护照进度更新了：”您的新证件已完成制证，可以领取。如您申请了回邮服务，我馆将尽快寄出，请注意查收”

六月二十一号 - Amazon Prime Day

  昨天是父亲节和老婆的生日，我们没有怎么庆祝，就是在家吃了个火锅，老婆不敢大肆操办是因为这个月的6万餐费挑战活动，貌似是完不成了，因为昨天过后还有3千的额度，但是距离月底还有10天左右。我倒真的没有把这个活动看的太认真，觉得能做到固然很好，但是如果做不到，也起到了节约的效果，也不错啊！但是老婆这么要强的人可不是轻易认输的人，她还在咬牙坚持说可以达成。这个月的现金活动其实对我们来说挺好的，以前刷卡的时候根本没有任何的感觉，就是觉得数字在流动，从我们的账户流到商家的账户，内心没有丝毫的波澜。但是自从本月开始了现金主义，看到纸币从指间溜走，还是切实的感觉到了痛苦。
  昨天给爸爸打了电话祝福父亲节的时候，两个孩子都很卖力的表现，让二老都很开心。
  今天是Amazon的会员日,好多活动齐发,搞的有点不知道买什么好了,但是我们列表中东西还是必须要购入的,至于野营用的帐篷-睡袋-桌子-椅子等,还需要好好的考虑一下。
  看了阿福的减肥成功的视频，对我触动还是蛮大的。他没有讲什么大道理，就是陈述了自己的经历：健康是一切后续行动的基础，不要拿什么工作或者家里的事来给自己找借口，因为如果身体不好了，生病了，反而会给工作和生活带来很大的打击，与其悔不当初，倒不如亡羊补牢，犹未为晚。CICO原则理解起来很简单，但是要自律并且长时间的坚持却很难。在日本还好，几乎所有的事物都会标明卡路里含量，只要控制好吃进去的卡路里小于消耗的卡路里，瘦是肯定的事情。如果再配合上运动，那么回归魔鬼身材不是梦了。

六月二十四号 - 开始瘦了

  今天称体重终于到了94.9，虽然下降的不多，但是对于减肥中的人来说还是蛮开心的。一定要继续坚持下去啊，16-8轻断食+早晨的健走练习，相信离真正瘦下来的日子不远了。加油！今早健走的时候，小腿肚子是有些疼的，差一些没有完成7000步的目标。但也坚持下来啊。减肥这个事情是没有回头路的。
  早晨是挣扎着起床的，有2次不起了的念头，但是还是决定翻身起床完成既定的计划，这才是第二天啊，我就开始了动摇的念头，坚持真的是不容易！昨天晚上11点入睡，虽然花了点时间来入睡，但是相较于12点才到床上，也是一点点的进步。早睡早起这个祖训，说起来容易，做起来真的难！
  今天乔总要走了，大家在食堂里聚了一下餐，然后乔总请大家喝了咖啡，我们也没有客套，直接喝着。叶总已经拿到了永住，杨总在申请中。到今年11月份我才满7年，慢慢来吧，但是总觉得没什么用。就是省的去更新在留卡了。

六月二十八号 - 瓶颈期

  周末这两天没有瘦，感觉遇到了瓶颈期，今天早晨6点起床继续去健走，完成了6公里的健走。回来的时候小腿稍微有点疼。16-8进食法依然在继续。

五月十七号 - 又懒了

  上次更新还是4月十四号，人一懒真的是什么都不想做。就想着喝点小酒然后微醺的睡去。那种感觉固然很好，但是会让时间变得奇快无比，感觉还没有过，一个月就悄悄的没有了。
  今天轴劲儿又上头了，就是找不到feb_in_japan.jpg这个图片，明明在github上可以看到图片，但是本地的文件中就是搜不到，真是费了好大的力气，最后发现是Firefox的缓存捣的鬼。太可恶了！🤬
  毛毛已经开始了全天的幼儿园生活，一切似乎都进入了正轨，我也得以继续开车上班，以前我是那么的热爱乘坐电车上班，但是自从开车上班后，渐渐的不想再坐电车了。开车的自由度很高，回家很快，而且还没有感染新冠的风险。
  老婆想兼职打工的心愿已久矣，但是找了一会儿发现并不是那么的好找，新冠肆虐的情况下，很多日本人都已经下岗找不到兼职，更何况是一个日语不是很好的外国人呢。慢慢来吧不急。

五月十九号 - 情绪控制

  今天跟约翰打字打了聊了很多工作的内容，感觉他的风格就是不停的输出，输出，在输出。完全不管接收者的反应，如果是以前的我，肯定无法控制自己的情绪跟他刚起来了，但是今天的我把这些归于工作，工作的事情就用工作的语言来处理，不可以夹杂自己的感情，出现问题就解决问题，如果解决不了就上报，不需要把自己的情绪搭进去。
  已经37岁了，才慢慢的可以分清楚什么可为，什么不可为。
  出门打水的时候遇到了長澤さん，真的是好久没有见面了，她刚好抱着一堆文件来HR办事情。她上来第一句话就问我：你现在可以说日语了吗？哈哈，我可以说的日语比以前好了很多，就用日语聊了一会儿，说了说孩子的近况。她住在春日部ークレヨンしんちゃん的故乡。她是我在UD认识的为数不多的几个日本人 。
  听说老钱的父亲因为新冠去世了，好可怜。总之印度的第二波疫情爆发，感觉让以前遥不可及的病毒已经蔓延到了自己认识的人或者相关的人，不敢想象如果我们一家四口如果被传染了怎么办，连个亲戚朋友都没有，真的是更加麻烦。所以一定要小心小心再小心。
  姥姥姥爷又给大绒子1000元过生日，还没有想好买什么，但是她马上就要8岁了是个不争的事实啊。逝者如斯，不舍昼夜！

五月二十一号 - Focus

  今天通过微软的Cortana系统定制很多的Action在日历上，比如2个小时的Focus时间，每天2次的Relax时间，还有下午的集中处理消息的时间等等。觉得还是很有用的，因为我总是放纵自己在一些没有计划的事情上，容易分神，如果通过日历可以不时的提醒自己该做什么也是很好的。而且可以同步TEAM上的状态，真不错。
  丈母娘六十岁生日，给她们预定了天津香格里拉酒店的至尊大床风景房，希望她们可以放松一下，度过一个难忘的生日，没有孩子和老人的羁绊，她们本就是应该享受的时间，但是她们太执念在钱方面，还在挣扎着让我们退订，我们坚决不同意！人的一生，说长也长，几十个年头，说短也短，可以做自己真正想做的事情的年头，屈指可数，尤其是在退休之后，如果还是不去体验自己从没有体验的东西，岂不可惜来人间转一圈。
  趁着午休的时间，看了一些Youtube的视频，有一个在日本做播音主持的中国姐姐，她分享的学习日语的秘诀就是要热爱自己学习日语这件事情。不要把学习当作痛苦的事情。别人问她学习日语最大的成就感是什么，她说：我把日本人都比下去了，就是比日本人说的日语还要好，一种不服输的精神吧。其实，如果问我“住在这里的理由”，我也说不出很多，但是如果想长久的住在这里，日语还是一个越早学好，受益越多的事情。

五月二十四号 - 又走了

  昨晚收到大姨夫走了的消息，虽然不是特别的熟络，但是听到这样的消息还是难免伤神的。这已经是癌症夺走的亲戚里第三个了。关键是从发病到走，竟如此之快。
  明天就是我大绒子的8岁生日了，预定了蛋糕，amazon购买了礼物，是一个多功能的麦克风，她跟妈妈都比较喜欢唱歌，希望这个礼物能够满足她们母女😄
  昨天跟晗晗妈妈念叨说护照过期办不了，自行车每月缴费但是不用等问题，结果人家回家后就帮忙搜索了解决方案，而且今早试了试居然真的预约护照成功了，而且老婆也下定决心要处理这个一直不适用还产生停车费的自行车了。感觉以前很多悬而不决的事情，如果上心去做，其实也是可以很快解决的。

五月二十六号 - 大绒子8岁了

  昨天给绒绒过完了8岁的生日，看不出什么变化，还是那么的不着调和无厘头，但是看着她俩在一起疯，一起搞恶趣味，还是会跟妈妈一起，看着她们，微笑。感叹世界对我们不薄，赐予我们这么好的一对儿女，这是多么好的一段人生体验啊！我总觉得，人的一生就是为了体验，体验一下前世时自己选定的下一世人生。所以无论如何，都是自己的选择，一定有自己的目的才会那样子选择，所以我们要做的就是遵从自己的内心，尽力的过好这一世。
  最近一直在听香港首富李嘉诚的故事，还是很有感触的，尤其是他对于两个孩子的教育方面，堪称典范，有太多的富翁以为给孩子最好的就是对他/她好，其实真的是南辕北辙，真正的对孩子好是对他们负责，给与他们正确的，潜移默化的影响，让他们在不知不觉中走好自己的路，这个需要很大的人生智慧来指导，很多富二代过不好这一生，其实只能是怪他们的父母没有用心的替他们着想，只是一味的给与孩子最好的物质条件而且还是试图留下不菲的财产给孩子，就算富二代可以过好他的一生，但是富二代会怎么教育富三代呢？肯定是如法炮制，但是有多少财富经得起这样的玩法呢？富不过三代是有道理可循的。
  前几日老婆说想尝试在6月份只花现金而不要刷卡，生活费暂定6万。虽然我知道是不可能的任务，可是我不能打消她的积极性，要支持她！比起现金消费，人在刷卡的时候，脑子是不会痛的。所以信用卡虽好，但是它会麻痹我们消费时候的神经，认为消费就是钱这种数字的转移，丝毫无感。
  刚才给家里打电话，爸爸点播我说：是不是最近很忙啊，来电话的次数少了呀。昨天绒绒生日的时候我应该给爸爸妈妈去电话的。但是又怕爷爷奶奶没有什么表示尴尬，所以作罢。其实杨晨在母亲节给姥姥钱，生日给姥姥预定酒店，我觉得都是应该的行为，丝毫没有觉得有什么不对，为什么不对爷爷奶奶那样。生活中事情都是无法太认真的，认真就输了。其实只要家里人高兴，这才是最重要的。

五月二十八号 - 惊魂一夜

  昨天晚上绒绒不知道抽了什么风，夜里2点开始乱喊乱闹，把妈妈和弟弟都吵醒了无法再睡。妈妈被气的发抖，深夜嘶吼，哎！养孩子真的不容易啊！

四月六号 - 樱花落尽

  今天听了ISUZU的老大片山さん的Townhall大会，他说了很多，从他的语言中我感受到了他期待变革的决心，也多少感觉到了跟Volvo联合，他们也多少看重volvo在技术上的优势，但是不是很确定telematics是不是他看重的一个点。其实高崎さん离职后的下一个公司也是做telematics的公司，从中可以多少看到telematics在未来的10年间是有很多的机会的，关键是看我能不能把握住。

十月六号 - 毛毛又感冒了

  小男孩是真的弱啊，上次感冒刚好没多久，这又感冒了！每次小孩子病之后就是父母的艰难时刻，他总是缠着你，让你抱着，还特别的矫情。2岁的孩子，没办法，他又不会表达不舒服，只是本能的发泄自己的难受的感觉。

十月十二号 - 月已过半

  老婆的微信终于可以可以正常登录了，这次换手机最大的问题就是微信应为微信的安全检测，限制了她的微信在3天之内无法登录新的设备。这3天确实证明了微信的威力，无法使用可是真要命啊，现在我觉得惩罚中国人的最佳方法就是让他在N天内无法使用支付宝和微信，真的是寸步难行吧。在我中华大地上，如果不让用这两个支付手段，无法买小吃的，没法下馆子，没法坐交通工具，没法酒店等等，最主要的是这些日常的地方不会为你准备零钱。
  通过这次在docomo办理新的iphone合同+办理金卡，貌似正式进入了docomo的经济圈，要不是乐天不给力，也不至于让我跳出乐天经济圈啊！竞争真的是好事，让用户自己选择，如果你怕用户流失，那你就要努力的提升自己，乐天在这次的手机支付战争中已经输了好多，估计会一蹶不振。

十月十六号 - Stress Check

  近年底的时候，各种测试都来了，首先是体检，然后是精神压力测试。应该问题不大！
  昨晚绒绒又一次无法入睡，被妈妈狠狠的说了一顿也没有从根本上解决问题，可怎么办啊？！
  最近发现跟绒绒的交流越来越困难，她总是想用日语表达，但是我跟妈妈又跟不上她的节奏或者误解了她的意思，让她很容易生气，我觉得还是应该好好把日语啃下来，不是为了别的什么，主要是为了跟女儿可以好好的沟通，无论是中文还是日语，作为父母的都要深深的理解她想表达的意思。

十月十八号 - 不是很忙的一天

  最近发现组里的人工作效率非常的地下，就是你不催他，他就天天在家上班，没有人知道他在家干什么。但是无论大事小事，你只要是问，他肯定没有完成，然后编出一堆理由。
  今天学些了日语语法里的“差一点就***”， 有两种表达方式： 1- ところだった　2- そうになる　車に轢かれそうになった　差一点被车撞了，学习日语是一个特别奇怪的感受，又会又不会的感觉，因为汉字的缘故，很多的东西看过去是明白意思的，但是不会用日语的发音读出来。想跟人聊这个话题吧，也无法聊下去。

十月二十号 - 星期五

  昨天晚上爆发了有史以来妈妈凶绒绒最激励的一次，妈妈同时哄两个小孩子睡觉，绒绒又犯病了，自己纠结于自己无法睡着的情况，乱踢，下床求妈妈抱等动作触发了昨天已经很累很累的妈妈的咆哮，吓得不轻。然后妈妈要求我们每个人哄一个孩子，替她分担一下哄睡的工作。
  以后在Amazon买东西，最好使用妈妈的手机D払い支付，可以获得3倍的积分。
