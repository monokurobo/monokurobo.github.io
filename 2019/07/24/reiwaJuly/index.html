炎热开始的七月

发表于 2019-07-24 分类于 碎碎念

七月二号 - 半年已过

  2019年过半，回头看看达成的”成就”,似乎没有。从1月26号搬进新家，就开始忙活，孩子的身份证，护照，Mynumber，役所的各种手续。目前来说，一切都已经上了正轨。也不能说什么也没有做，但是这些都是应该做的。
  一切都顺调之后，遇到了一个问题，不知道该往哪个方向努力了。最近的一年，对IT技术已经生疏了很多，更多的是关注测试方面的计划，统筹等等。随着跟各个部门打交道，自己的心性和交互能力提高了很多，知道了什么可为，什么可以变通，什么可以拒绝。但是隐隐的有些担心，如果明天我被通知不要来上班了，我可以找到别的工作吗？
  人的发展有很多中可能，为什么我选择了测试岗位？是觉得技术开发已经没有什么意思了。我想继续转管理的岗位，如果问为什么，首要的是薪水会提高，然后才是其他的原因。没有必要不说出心里的真实欲望，我也知道，管理的岗位有管理岗位的苦处，不能光看贼吃肉，不看贼挨打。

七月三号 - 尝试VSC

  VSC吸收了很多Sublime的很多优点，而且直接集成了Terminal，用起来还是很方便的，尝试使用一下。

七月四号 - 幼稚园参观日

  今天上午在家里办公，中午去参加了绒绒幼稚园的音乐课和英语课，还是蛮有意思的。录了很多的视频，接绒绒回家之后都没有机会给老婆看，就赶紧来公司上班了，工作原因+找同事帮忙带丈母娘的签证申请文档回天津。总体来说还是很顺利的。加油！

七月十号 - 普通的一天

  今天是第一次在VSC上写记录，特此记录一下。最近我开始焦虑，集中在我对自己目前的能力没有信心，总是想着如果明天UD让我回家，基于我现在的能力，是否可以快速的找到下一家。因为我是一个有贷款的人，我开始想着要学好一方面，然后作为自己的B计划，就算发生意外，凭借我现在学习的技能也可以让我平稳的找到下一份工作。
  目前我把学习的目标放在了python上，首先是因为他的流行程度越来越广，在未来，一切都是数据为先，基础建设都已经交给了云计算和自动化。我认为，未来对于数据分析的职位会很多，所以多学一些技能或者多实践一些，对未来是有好处的。

七月十六号 - 没有生病的一天

  三天小长假过的真的是惨不忍睹。绒绒在幼儿园被”咽頭結膜熱”感染了–俗称プール熱，我开始专门负责照顾她，跟上次流感的情形类似，好不容易她复原了，我和老婆都开始出现了不适的症状，我还好，持续了一天，今天可以正常上班。老婆还没有复原，各种不舒服。
  人就是这样，拥有的东西从来不知道珍惜，比如健康。当大家都好好的时候，没有人注意到这也是我们所拥有的东西。但是一旦我们失去她，才会意识到她的珍贵。

七月十七号 - 没有生病的一天

   工作还算顺调，但是在思考要不要去开发那里插一脚。深知，如果跟领导说了，她肯定愿意，多出来的劳动力白要谁不要呢。但是对于我来说，必须要在完成本职工作的前提下，才可以从事开发工作

七月二十二号 - 晚上出去和咖啡

  昨晚破天荒的跟晗晗的爸爸一起出去喝了咖啡，他是个不善言谈的人，但是我也从他很少的话里得到了很多对我有用的信息。比如

  1. 其实，不是为了喝酒而喝酒，不是为了抽烟而抽烟，只是借着这些机会去跟别人谈一些不能在办公室谈的的东西。
  2. 我在抱怨印度人的一人得道鸡犬升天，他的答案让我很意外，他说：印度人都是修双学位的，不但修了理科，而且还修了管理学或者MBA等等，所以我们中国人跟人家比的时候，虽然技术占优，但是管理不足的短板是限制我们不能像印度人那样当管理层的原因。
  3. 他劝我都MBA换一个角度看待这个世界，而我的第一个回复是：我的日语不行！真的小孩子气的回答，如果一切都是困难为先，世界为什么善待你呢？

我觉得这次的谈话对我触动挺大的，我还在自己的安乐窝里恣意，人家已经在下一个跑道上飞奔了，时不我待，学习日文去！

七月二十四号 - 难得的休整

  忙了这么久，终于敲定了可以部署了，我也算是松了一口气。今天跟大领导聊天，汇报了BA的不专业，领导估计会辞退这个不称职的人，UD付了那么多钱给一个不专业的人，而且他还给项目带来了很多麻烦，真的应该早点这么干!
  关于自己的职业规划，从B到C的这个过程，真的需要自己的更加努力才可以。

梅雨六月
