美丽的秋天

发表于 2019-10-01 分类于 碎碎念

十月一号 - 新的月份开始了！

  早晨时候就开始查看绒绒幼儿园的园费收取情况，就想看看政府到底给减免了多少，结果还是蛮喜人的:减免了2.5万左右，如果按照增长2%的消费税来看，我要消费125万的日币，才会是2.5万日币。由此可见，日本政府的执行力度还是很给力的。我们其实是这个项目的实际受益人。绒绒可以用几个月，毛毛倒是可以从头适用。如果每个月可以减少2.5万的支出的话，对于我们的生活质量还是会有帮助的。
  今天是国庆，二姐出游，大姐夫加班，只有哥一个人去爸妈那里过节，妈还说猪肉已经涨到22块钱一斤了。呵呵！
  从10-12月份，绒绒的校车改成晚bus，不需要我送了。顺势开始了我的跑步健身之路。带着我新到的无线耳机，7点40开始跑步，30分钟3.6公里，跑完之后整个人都精神了，很爽！见面打招呼就可以感觉的出来。怪不得日本人见面都会问：”お元気ですか?” 看来人的元气/精神头真的很重要，今天还搜索了一下成人每天的卡路里摄入量。争取控制在1800-2000，这样坚持下来的话，身体的整体状态是会变好的，减肥不减肥倒不是很重要，还是要看BMI和体脂率。
  Hexo的日志按照更新日期排序的问题解决了，告别了手动的方式，开心一下。参见这里

十月二号 - Package Syncing

  随着设备的增多，软件的配置和偏好同步问题逐渐成为了问题，比如自己常用的SublimeText，目前还只有2台工作电脑的我就很苦恼他们之间的同步问题，有的时候在公司的电脑配置好了，回家以后又不能如预期的工作了，今天参照这里完成了Sublime和Google Drive的同步。这样就可以在任何机器上快速得到自己常用的package然后开始工作了。
  今早突然告诉绒绒我要带她出去玩，而且每个月都有一次机会，她虽然极力克制自己的高兴，但是在我出门的时候，她还是跑过来，异于寻常的跟我告别，从这里我知道了，我不是一个好爸爸，跟她互动的太少了。现在我要把这件事写到每月必做的todo list。

十月十号 - 看病归来

  入院->治疗->出院，三天的流程把我们折腾的够呛，但是无论如何，总算是安全无虞的结束了，一切都是好的结果。中间还穿插了绒绒的小学入学体检。发现她的眼睛有点异常，需要进一步诊断，其他一切都顺利。毛毛回家后就开始爬，玩，跟没有任何事情发生一样，让我们对日本的医疗技术有了全新的认识，而且整个手术一分钱都没有花，全部由健康保险负担了。真的是很省心。

十月十一号 - 台风19号将至

  周末的台风来势汹汹，铁路，商店，医院都已经宣布要停业。超市的干粮被抢购一空，种种迹象都表面这次的台风非同寻常，而且要做好停水停电的准备，现在唯一不希望停的就是网络了。被困在家里出不来，如果连网都无法上的话，真的是要无聊死。从这从表征来看，网络在我心中的地位似乎已经超越了水电等生活必须设施。真的挺可怕。
  今天是正式使用Office365的第一天，我的电脑版outlook出现了问题，只能使用web和手机版，但是使用体验也挺好的，不输电脑。现在真的是什么都上云了，以前觉得不能想象的东西，现在都已经实现了，AWS已经接管了大多数的网络部署，现在微软的云把操作系统也给上到了Azure上，以后真的是不需要安装任何东西了，因为一切都存在云端，只需要用户名和账户，所有的设置都会被推送过来。便利是便利了，但是总觉得隐隐有些害怕，如果大家都依靠云服务，如何确保云的持续性和稳定性就很成问题了，比如前些日子，AWS宕机就导致了N多大型的服务和网络瘫痪好久。如果有人能够控制Amzon和Azure，后果不堪设想。

十月十五号 - 新的一周开始

  3天小长假匆匆而过，真的感觉就是嗖的一下子。周末的时光似乎都是为了家里人在忙碌，因为他们默默地生活了一周，需要周末的时间来体验不一样的日子，带着他们逛商场，买东西，看病，看孩子。

十月十七号 - 孩子又要病

  昨天开始，两个孩子相继出现了发烧和流鼻涕的症状，这是要病的节奏啊。每次到了天气转凉的时候，都是得病的高发期，就连姥姥也是老有鼻涕的感觉。生活就是没有消停的时候，忙完一波活儿，就会再来一波，总是让人有的忙。这也是为什么在有了孩子之后，感觉日子过的就是一瞬间。因为孩子的大事小事填充在每天的日子里，让人感觉不到自己做了自己喜欢的事情，总是被孩子的事情推着走，今天预约了看牙，改天是看眼睛，周二游泳，周日跳舞，凡此种种。真的很让人心烦和抓狂。但是每每要崩溃的不行的时候，突然瞥见孩子的笑靥如花，一切的辛劳立刻忘记。真的是立刻忘记，就是想赶紧过去抱抱她/他。如果我们生活的世界真的是虚拟的世界，那我真的很佩服系统设计里关于父母和孩子的这套设定。设计的如此完美，天衣无缝。让父母心甘情愿的为孩子付出，不求任何回报。就是因为他/她继承了你的部分长相和性格。佩服的五体投地！

十月二十一号 - 孩子病了好几天了

  现在两个孩子都开始了咳嗽，大的那个咳的晚上无法好好的睡觉了。没办法，只能去医院看医生了。本来以为靠孩子的抵抗力是可以扛过去的，看来还是不行。
  周五中午，带绒绒去看牙齿矫正，医生可以说英语，这次是真真正正的弄明白了：7-8岁的时候是第一阶段，但是无法保证可以治好，因为随着孩子身体的发育，在14-15岁的时候才会达到巅峰脸部骨骼停止生长。每个阶段都是40-45万日元的左右，唯一的区别就是，如果第一阶段做了，有10%的机率让孩子在第二阶段自愈。80%还是要在第二阶段矫正，最后10%是通过手术，但是概率很小。老婆还是想做，我不是很想用40多万去拼10%的自愈概率。让她跟姥姥仔细商量吧。
  周六上午，老婆带孩子去检查眼睛，结果用了足足4个钟头，可把人给急死了。老婆说：以后再也不去大医院了。
  原来不光是我觉得周末更辛苦。

十月二十三号 - 老大可以去幼儿园了

  昨天我跟绒绒在家，姥姥+妈妈+毛毛去costoco买东西，我们俩是逍遥自在啊，但是他们被堵在超市门口从10点-15点，足足等了5个小时才进场购物，晚上8点多回来的，累的要死！
  绒绒白天不是很咳嗽了，终于可以去幼儿园玩耍了。妈妈也可以松口气了。这个月查工资，发现被扣了3千多，应该是有一天没有完成8小时导致的，这个月要好好的努力！

十月二十五号 - 又到月底了！？

  这个月因为有很多的事情发生，所以感觉过的是特别的快！好的一方面是，一切都是好结果。这个周末预约了軽井沢的温泉酒店，让全家人都可以散散心，透透气。
  这个月的工作不是很忙，因为很多项目都是在开发阶段。从下周开始就要忙起来了。我尝试找些自己感兴趣的来玩，比如node和mongo的测试代码。但是写着写着，找不到了当年的激情，感觉就是在重复劳动的感觉。然后跟我姐也表达了我在工作上失去了方向，她劝我要趁年轻，争取坐到领导的位置。跟天津的付姐也聊了一下，她劝我发挥我的多语言沟通过的优势来寻找机会。我觉得我需要充电，但是充电的内容还没有确定下来。

十月二十九号 - 旅行归来

  周末和周一两天，去軽井沢玩耍了两天，旅途还是很顺利的，该去的都去了，该玩的都玩了，绒绒也对这次旅行进行了高度的评价，说这次的旅行是最开心的一次，因为被酒店的寻宝游戏给留下了深刻的印象。我计划自己给她设计一个这样的游戏。
  这次短途旅行，已经是尽力的选择不累的行程了-购购物，吃吃自助，逛逛公园，泡泡温泉，但是还是感觉累，人老了以后怎么那么弱 ！尤其是带着两个孩子的旅程，小宝儿在车里不停的哭，估计是晕车了，但是还是很影响开车心情的。这次的来回都是我开的车，300KM的距离，我开始喜欢开车的感觉了。
  今天干了一天的工作，很少见的日子，而且中间还代购了一次，并且给付姐讲了一下VDP，充实的不行！

十月三十号 - 最后两天

  过着过着，日子就没有了。今天找同事要了两张东京车展的票，准备跟绒绒去见见世面。姥姥回程的准备工作已经开始了。这次我打定主意，不让丈母娘出任何的钱，给她买的任何东西我们来买单。
