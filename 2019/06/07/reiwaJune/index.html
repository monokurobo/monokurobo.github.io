梅雨六月

发表于 2019-06-07 分类于 碎碎念

六月三号 - 继续工作

  周末的肠胃病毒发作，大部分时间都是在厕所里度过的，肚子都被拉空了。强撑着来公司上班，中午就在Lawson买了一份面条，不敢多吃，因为吃了的东西马上就会排出去。我是怕了！
  这已经是很恶心的事情，但是吃饭的时候听到了更加恶心和荒谬的消息，石头居然要当架构师了，我的天。他当个开发人员都不够格，屁都不会的一个人，居然去当架构师，真的是亮瞎了我的氪金眼。最关键的是，领导们居然也同意了，就由这他这么瞎搞吗？当时想着，这他妈的太扯了，如果他能胜任，我觉得便利店的大爷也可以的。一点相关培训都没有参加过，就去当架构师了，UD的人士任免真的是玩笑。其实这本来就不关我的事情，人家爱干什么干什么，又没有影响我的收入。

六月七号 - 难得的周五

  紧张的4天过去了，每天都跟打仗一样。今天抽时间搞清楚了投资信托，原来我之前照猫画虎的乐天薅羊毛不是买的股票，是买的投资信托，而且必须是積立注文才可以使用信用卡支付。第一次的支付和赎回已经结束，赚了680，算是第一次尝试吧。第二次估计要等月底再卖出吧，本来就是想薅羊毛赚取一些积分，没想到资本市场还是很好玩的，以此为入口可以继续理财生活。
  最近很多的大龄明星公布婚讯，大家都觉得不般配，说好白菜被猪拱了，看了日本那一对，男方是搞笑艺人，在新闻发布会时，有记者故意刁难说他爱人是心机女，他不卑不亢，分分钟霸气控场，最后一句话还能把快哭的女方哄回来，这份淡定从容，情商爆棚的男人，如果我是女的，也会义无反顾的嫁了。年轻的时候看颜值是对的，因为荷尔蒙作祟，都会挑选自己看着舒服的，这无可厚非。但是三四十的时候，就渐渐明白了，能找到一个三观合适，有话说，呆在一起舒服的人长久的过日子是多么难。吃瓜群众真的够累的，人家结婚，哪容得你们这些妖怪反对！

六月十号 - 梅雨开始

  一直欣赏不了雨的美，总是让我高兴不起来。这个月的主要任务是给女儿买书包，书包很贵，但是可以用6年，所以平均到每一年的花，还是可以的。老婆的选择困难症又发作了，一直催着赶紧搞定。

六月十一号 - 突然晴天

  本以为梅雨要下一个星期，结果今天居然停了还出了太阳。还挺舒服，伴随着好天气，我的测试工作也终于有了一些进展，终于可以在VDP上看到数据了，这是忙活了1个多月的结果啊！这个项目结束就再也不想跟这个BA合作，太他妈的费劲了。

六月十二号 - 希望今天测试可以全部搞定

  毛毛越来越可爱了，每天不看到他还真的不行。我们昨天开始反思：我们虽然都已经达成了共识- 不凶绒绒，但是事情发生了，两个人都控制不了自己的情绪，有的时候妈妈说的时候，我更凶的说她，让绒绒无地自容，如果在家里都要不停的道歉和逃避说教，到了社会上对她来说更是一种可怕的未来。所以从今天开始，我要开始记录自己发火的次数，不能保证一次没有，但是要尽量减少，一直到没有为止，就像戒烟一样。

六月二十号 - UAT结束

  终于结束了UAT，我的工作就可以告一段落了。下周开始在家工作。今天是老婆的生日，33岁的老婆没有奢望什么，就是简单的定个蛋糕，本来像出去奢侈的吃一吨的，但是鉴于两个孩子的身体情况，决定在家里吃披萨。老婆变的越来越会过日子了，看到这些我应该是高兴才对，但是总是觉得不舒服。她跟了我之后只能让她的欲望变低吗？不可以，所以我觉得把去台湾的旅行的计划作为礼物送给她。不能因为这样那样的原因不去做一些这个年纪应该做的事情。

六月二十四号 - 删除游戏和资讯APP

  今天看了一篇订阅的文章，看完以后，句句扎心，为什么我不能跑过大多数人，很大的原因是我没有危机感，有长辈的照拂，不需要担心老家年事已高的爹妈，还有时时刻刻能能帮助我的姐姐，这就让我产生了一种老婆孩子热炕头，不思进取，只求安稳的错觉。每天就是正常上班，路上玩会手机游戏，刷刷新闻，微博啥的，回家照顾孩子直到睡觉，孩子睡后，跟老婆一起追日剧。每天都过的一样一样，所以一晃2019年就过去一半了，而我的计划一个都没有实现。
  意识到了错误的下一刻就需要改正，立刻删除所有手机里的游戏和新闻，乱七八糟的APP。在没有的情况下，可以听听歌，听听日语广播，看看电子书也是好的，不能再把自己的碎片时间用来看那些没有用的东西了。自己感觉充分利用了碎片的时间，其实自己是被碎片时间绑架了，做着重复的动作，最后沦为每天重复的移动流量入口。
  工作不是很忙，就搞了一下：ふるさと納税。以前听日本同事介绍过，但是听着那些手续就觉得太麻烦了，也就没有动过念头，现在听说手续已经简单到3步，所以就尝试了一下，花了1万日元买了10KG的秋田大米，估计要到11月份才会发货，在这1万日元里面，我需要自己负担2千日元，剩下的8千可以抵扣我明年的住民税，话说现在的住民税真是越来越高了，因为我选择了年金的提前支取，所以导致我的到手收入就高了，没有了年金的免税效果，我就要面临每年的住民税的提高。为什么要提前支取？ 因为购入房产借了些钱，所以想着可以早点还钱。明明知道是不对的，但是欠债的感觉不好，没办法。

六月二十六号 - 没有游戏和微博的日子

  今天是没有游戏和微博的第二天，没有感觉明显不适。上班的路上听了NHK的广播，大部分都没有听懂，只能抓取一些关键词，还是要持之以恒啊，昨天回到家后，没有游戏和微博的牵绊，我不再趁着间隙去拿手机刷刷刷，而是可以从容的做好每一件事情，这样的感觉挺好的，我喜欢。
  最近我有了一个新的小任务，就是让自己的脸不再有疙疙瘩瘩的苞，不再流血，我想有一张干净的脸，不为什么，就是想干干净净的活着。

六月二十八号 - 没有游戏和微博的日子继续

  今天是没有游戏和微博的第二天, 渐渐习惯，但是上厕所的时候还是希望拿着手机，哪怕不玩游戏和看新闻，也想着看看youtube啥的，真的是不好戒。
