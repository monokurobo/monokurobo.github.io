繁忙五月

发表于 2019-05-14 分类于 碎碎念

五月七号 - 长假归来

  10连休归来，堆积的工作要开始清理了。天气开始慢慢变热，夏天来临，上班的路上，湿漉漉，汗涔涔的感觉又要来了，好讨厌的感觉，没办法啊!
サラリーマン的宿命。
  在这十天里，去赏了紫藤，买了玩具，玩水，动物园，快乐堡，绒绒的活动倒是挺丰富的。可怜了妈妈和毛毛在家死磕了好多天。每天一有空我就找妈视频，让她在无聊的时候看看孙子，她也很开心的样子。

五月九号 - 二宝坠床

  其实是昨天晚上的事情，妈妈把二宝儿放在了床的最里面，然后去准备毛巾给他擦脸，就在这么短的时间里，他爬到了床边，就在妈妈回来的一瞬间坠落，妈妈用手垫了一下，但是头还是着地了。把妈妈吓了个半死。当时的我也是蒙的，不知道该怎么处理，不知道有没有摔倒脑子，我只好尽力的告诉自己，不要埋怨妈妈，不要惊慌，冷静，冷静，再冷静。结果是好的，经过了一个小时的观察，二宝是没有问题的，一如既往的顽皮。我们就稍稍放了一点心。
  高兴的事情就是我们惺惺念念了好久的运输重物的车终于到货了。各方面都很满意，好收好放，使用简单。
  工作上两个项目一起干，多少有点烦乱，但是没办法，这就是工作，即使不想做，也要做的事情。今天早晨对Vhio发了一顿火，上一次发火还是跟石头吵架。没想到年纪长了那么多，还是没有控制好自己的情绪。

五月十三号 - 继续工作

  E2E测试就是一个不断发现幺蛾子的过程，你不知道都是什么问题在等着你，一会有人误删配置文件，一会因为打错一个字母而导致系统不能运行，这的是磨练人性的一个好机会。

五月十四号 - 开始降温

  今早突然飘起了细雨，很舒服，但是我不太喜欢下雨的气氛，总是想抽烟，不知道这是一个什么生理反应，也许是在瑞典出差的时候，我的身体记忆了雨天抽烟的形态，所以每逢下雨，不自觉的就会提醒我烟的味道，人的身体真的奇妙的很。昨天妈妈在nitori买入了在墙上贴的贴纸，她知道自己是火命，所以养不活真的植物或者花，索性就买了3盆假花贴纸。但是效果出奇的好，瞬间让我们的家温馨了好多，还有照片墙，生活虽然不易，但是只要有装点他的心情，一切还是没好的！
  Line的Lawson炸鸡居然中奖了，开心就好

五月十五号 - 磨练耐性

  今早Suda又给我找另外的活儿，被我给拒了。上次的经验还记忆犹新，为了弄那20个Units，忙的要死要活的，最后也没涝好。所以索性直接拒绝，还好我手里有其他的急活儿。学会拒绝也是成长的一部分。
从今天开始我要开始学习日语，目的就是-看书

五月十六号 - 家庭财务状况统计

  今天统计了一下可支配的日元情况并且发送了老婆，她平时不太关心家里的经济情况，没有任何的目的，就是望周知。一切还算是顺调吧。期待着七月的奖金。因为这个月绒绒要过生日，我们牺牲了五一出游的计划就是为了给她一个完美的生日。预约了酒店，准备了2天的行程。妈妈提醒我在迪斯尼乐园尽全力满足绒绒提出的愿望，让她过一个难忘无比的生日。我答应了，让她在迪斯尼的世界里做一天的公主，万般宠爱的那种。
  绒绒在各方面都是一个好孩子，仅有的一些小毛病也是可以理解的，毕竟是小孩子吗，跟同龄的小孩子比，总是领先他们太多。

五月二十号 - 非常忙的一周

  早晨跟Suda review了一下touch point，发现自己写的那些计划怎么那么好，而我平时总是偷闲去看一些乱七八糟新闻啥的，今年是我要在工作上进步的一年，不能满足于工作及时完成，需要更前瞻的制作计划和执行，多一些知识储备，更加高效的完成自己手里的工作，这样才有时间参加一些其他的活动。

五月二十一号 - 忙碌的一天

  项目期限将近，很多问题才浮出水面，这个BA真的是我见过的最差劲的consultant，没有最差，只有更差。没事就请假，没事就晚到，问啥都是恍然大悟刚知道的样子，操，我是测试经理，不是BA，服了！

五月二十九号 - 终于有点眉目一天

  二十五号的绒绒生日，绒绒是过爽了，先去迪斯尼乐园再去迪斯尼海洋，晚上住酒店。要什么就买什么。高兴的不要不要的，但是2天一晚的旅游一结束，两个孩子同时发烧了，老婆同时照顾他们两个很辛苦。因为项目时间紧的关系，我也不能在家支持，检讨。说着说着，五月份就要结束了!
